# Founder Rule #1

We do not build features first.

We build experiences first.

We do not ask:

"What should we build?"

We ask:

"What should users feel here?"

Cursor does not define the product.

Cursor implements our decisions.

When uncertain...

Choose simplicity.

Choose workflow.

Choose trust.

Choose fewer clicks.

Choose one obvious next action.

Never optimize for feature count.

Never sacrifice the operating loop.

Never build because competitors have it.
