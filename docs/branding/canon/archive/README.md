# Archive

Retired Canon assets. IDs remain reserved and are never recycled.

## Naming

Keep the original Asset ID from the source category (for example `ALT-WLD-001`).

## Approved asset set

Every archived image retains:

1. **Image** — the visual file
2. **Matching Markdown** — caption, usage, and metadata
3. **Prompt History** — generation and revision record
4. **Approval Notes** — review decision and date

Update status to `Archived` in `docs/branding/CANON_IMAGE_LIBRARY.md` when moving assets here.
